## Text 

Clarity is key. 

In the asthma response protocol for example we don't need the steps itemised (step 1, step 2) as it increases cognitive load. 


## Missing Links

Sometimes we have links that could be denoted using feint lines.

For example:

The yes response for Step 3 (provide the inhaler) then instructs to proceed to step 4. That can be denoted with a feint diagonal line leading to line 4.

## End of Path

Sometimes, a pathway reaches its conclusion

We can use two underlines to denote that

## Icons

The selective use of icons can make the information more digestible. The key is to use them judiciously.

For the administer inhaler step an icon of an MDI inhaler could be used.
